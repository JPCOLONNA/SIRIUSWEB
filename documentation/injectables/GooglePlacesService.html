autocompleteRequest
autocompleteRequest(input: string, address: boolean)

Requête vers Google Places API Autocomplete qui renvoie une liste de prédictions

Paramètres requis / préférés pour une requête:

  • input *: valeur à chercher [OBLIGATOIRE]
  • key *: clé API [OBLIGATOIRE]
  • types: type de lieu à trouver (address / postal_code / establishment / geocode)
  • components: permet de limiter à certain(s) pays (country:fr)

Documentation: https://developers.google.com/places/web-service/autocomplete?hl=fr Exemple de requête: https://maps.googleapis.com/maps/api/place/autocomplete/json?input=32%20Rue%20de%20Bellevue &types=address&components=country:fr&key=AIzaSyDUmo-X8fj0Gh9hbXUSdDDhhD3kE8dPDPU

Parameters :
Name Type Optional Description

address boolean
Returns : Observable<any>
